# Cara Force Refresh Aplikasi

Aplikasi sudah diperbaiki dan build berhasil ✅

## Langkah Force Refresh:

### Opsi 1: Hard Refresh Browser
1. Buka `http://localhost:5173`
2. Tekan **Ctrl + Shift + R** (atau **Ctrl + F5**)
3. Tunggu beberapa detik

### Opsi 2: Clear Cache Manual
1. Buka Developer Tools (F12)
2. Klik kanan pada tombol Refresh di browser
3. Pilih **"Empty Cache and Hard Reload"**

### Opsi 3: Restart Dev Server (PALING AMPUH)
1. Di terminal, tekan **Ctrl + C** untuk stop server
2. Jalankan lagi: `npm run dev`
3. Buka browser di `http://localhost:5173`

### Opsi 4: Incognito Mode
1. Buka browser dalam mode Incognito/Private
2. Akses `http://localhost:5173`
3. Login seperti biasa

---

## Jika Masih Blank:

Cek Console Error (F12 → Console tab) dan screenshot error-nya untuk saya.
